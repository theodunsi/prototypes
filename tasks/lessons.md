# Lessons Learned

## 2026-03-01
- **Always extract and study reference screenshots/video frames BEFORE building.** The initial plan description didn't match the actual design (different layout, colors, structure). Should have verified the reference visuals first rather than building from a text description.
- **Don't rebuild without confirming direction.** When the user says "it doesn't match," study the reference carefully and propose specific changes before rewriting everything.
